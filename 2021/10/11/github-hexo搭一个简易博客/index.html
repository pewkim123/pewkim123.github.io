github+hexo搭一个简易博客

发表于 2021-10-11

第一篇blog，简单记录下用github和hexo做一个简易博客的过程。

安装NodeJS

官网直接下载即可,没啥好说的。

注意下版本问题，太高hexo不支持，太低也有问题。

C:\Users\10247>node -v
v11.12.0

C:\Users\10247>npm -v
6.4.1

安装hexo

npm install hexo-cli -g

hexo发布博客

hexo init        //找一个空文件夹初始化
hexo new "title"    //创建一篇新博客
hexo clean
hexo s        //本地server
hexo g        //生成静态页面
hexo d        //部署到github

更换主题

打开_config.yml，修改如下

theme: next

注意空格。
